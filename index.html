3-1. DB入れ替え（フルリフレッシュ）SQL

SSMSの New Query で実行。クエリ窓のDBは master に。対象DBノードは折りたたむ。RDSは SINGLE_USER 不可のため KILL してから DROP。初回は DROP が自動スキップされます。


    

  


  

    
3-2. web.config 接続文字列

    
Web.config の <connectionStrings> 内の DefaultConnection 行をこの1行で丸ごと置き換える（SQL認証）。Trusted_Connection / Integrated Security は入れない（入れると NT AUTHORITY\ANONYMOUS LOGON エラー）。

[Unit]
Description=Apache Tomcat
After=network.target
[Service]
Type=forking
Environment=CATALINA_PID=/opt/tomcat-10/temp/tomcat10.pid
Environment=JAVA_HOME=/opt/jdk-21.0.11
Environment=CATALINA_HOME=/opt/tomcat-10
Environment=CATALINA_BASE=/opt/tomcat-10
Environment="CATALINA_OPTS=-Xms512m -Xmx1G"
Environment="JAVA_OPTS=-Dfile.encoding=UTF-8 -Djava.awt.headless=true"
ExecStart=/opt/tomcat-10/bin/startup.sh
ExecStop=/opt/tomcat-10/bin/shutdown.sh
User=tomcat
Group=tomcat
[Install]
WantedBy=multi-user.target
